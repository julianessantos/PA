//Assumindo que pulo[] é um vetor do tipo int, quais das seguintes expressões referenciam o
//valor do terceiro elemento do vetor?
//  *(pulo + 2);
//  *(pulo + 4);
//  pulo + 4;
//  pulo + 2;
//
// Resolução:
// 	A expressão que melhor referência é o valor do terceiro elemento é '*(pulo + 2);', Porque ele pega o conteúdo do vetor
// e posiciona no terceiro elemento, uma vez que os elementos do vetor são expresso dessa maneira:
// pulo[0,1,2], pulo[2] seria o terceiro elemento.
